# Visual APL
A web based visual scripting tool for writing World of Warcraft "Action Priority Lists" to be used with [SimulationCraft](https://www.simulationcraft.org/) / [RaidBots](https://www.raidbots.com/)

This project uses a React + Vite and React Flow and java script front end, and will utilize either a python or go backend for generating APL files.
